Noodle Project

Monitor your pet snake's environment and make changes from anywhere. You are in the right place to get a developer overview.

What Is It?

The Noodle Project is a system for controlling and monitoring the environment of a pet snake enclosure.

More Information

For more in depth technical information on the project, visit the wiki. The wiki has information on hardware and software requirements as well as how to get the project running.

Bug reports, feature requests, or technical questions should be added to the issue tracker.

If you would like to contribute, please view the README for information.
